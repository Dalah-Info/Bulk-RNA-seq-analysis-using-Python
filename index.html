✅ قائمة المهام قبل ورشة: تحليل بيانات Bulk RNA-seq باستخدام بايثون

🧰 هل لديك Python وAnaconda على جهازك؟

إذا كانت الإجابة نعم، يرجى:

  1. فتح Anaconda Prompt أو Terminal
  2. إنشاء بيئة جديدة: 
    conda create -n bulk-rnaseq python=3.10
  3. تفعيل البيئة: 
    conda activate bulk-rnaseq
  4. تثبيت الحزم: 
    pip install numpy pandas pydeseq2 scanpy scikit-learn matplotlib seaborn plotly sanbomics mygene gseapy notebook

❌ إن لم يكن لديك Python أو Anaconda

يرجى:

  1. تثبيت Python
  2. تثبيت Anaconda
    🎥 اضغط هنا لمشاهدة فيديو التثبيت
  3. اتبع نفس خطوات إنشاء البيئة وتثبيت الحزم أعلاه
